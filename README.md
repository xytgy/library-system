# 图书管理系统 (Library Management System)

本项目是一个基于 Vue 3 和 Element Plus 构建的现代化图书管理系统，旨在提供高效、便捷的图书借阅及管理方案。

## 核心功能

- **数据统计**：直观展示图书借阅趋势及库存概览。
- **用户管理**：多角色（超管、管理员、学生）权限控制及账号维护。
- **图书管理**：完善的图书入库、编辑、查询及库存维护功能。
- **分类管理**：灵活的图书类目划分与层级管理。
- **借阅管理**：完整的借阅申请、审批、归还及逾期追踪流程。
- **热门图书**：基于借阅频次的图书热度排行榜展示。

## 技术栈

- **框架**：Vue 3 (Composition API)
- **构建工具**：Vite
- **UI 组件库**：Element Plus
- **状态管理**：Pinia
- **路由**：Vue Router
- **通信**：Axios

## 环境要求

- **Node.js**: v24.14.1 (推荐使用 v20.x 或更高版本)

## 快速开始

### 安装依赖

```bash
npm install
```

### 本地开发

```bash
npm run dev
```

### 项目打包

```bash
npm run build
```
